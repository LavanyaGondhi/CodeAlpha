# Stock Portfolio Tracker
This is a simple Python program developed as part of my CodeAlpha Internship.

## The program allows users to:
- Enter a stock name
- Enter the quantity
- Calculate the total investment based on predefined stock prices

## Concepts Used
- Dictionary
- Conditional statements
- User input handling
- Basic arithmetic operations

## How to Run
Run the Python file using:
         python stock_tracker.py
